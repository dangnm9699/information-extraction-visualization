# Information Extraction Visualization

- Named Entity Recognition Viz
- Relation Extraction Viz

### 1. How to run

Prerequisite:

- Node: v16.13.0 LTS
- NPM: v8.1.4

#### 1.1. Run development

```bash
# 1. install packages
npm install

# 2. start development env
npm start
```

#### 1.2. Run production

```bash
# 1. install packages
npm i && npm i -g serve

# 2. build
npm run build

# 3. start
serve -s build
```
